# Auto Focus Keywords

Auto Focus Keywords is a WordPress plugin that automatically assigns SEO focus keywords based on page, post, or product titles.

It is designed to ensure consistent on-page SEO signals without manual repetition or editorial overhead.


## Purpose

Many WordPress sites rely on SEO plugins that expect a focus keyword field to be manually filled.
At scale, this often leads to inconsistency, omissions, or meaningless placeholders.

Auto Focus Keywords exists to stabilize this signal by deriving it directly from the actual content title.


## Core behavior (high level)

– Automatically sets focus keywords from titles.
– Supports posts, pages, and WooCommerce products.
– Integrates with common SEO plugin focus keyword fields.
– Operates silently without altering visible content.


## What this plugin is not

– Not a keyword research tool.
– Not an AI content generator.
– Not a ranking guarantee.
– Not a replacement for editorial SEO decisions.
